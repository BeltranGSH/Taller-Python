# Taller Python para la Ciencia, Tecnología e Ingeniería

## Descripción del Proyecto

Este repositorio contiene el trabajo realizado durante el taller de Python, cubriendo conceptos fundamentales de programación aplicados a ciencia e ingeniería. El proyecto incluye ejercicios prácticos sobre entrada/salida de datos, estructuras básicas, uso de módulos científicos (numpy, matplotlib, pandas) y ejemplos aplicados.

## Requisitos e Instalación

### Prerequisitos
- Python 3.8 o superior
- pip (gestor de paquetes de Python)

### Instalación

1. **Clonar el repositorio:**
\`\`\`bash
git clone https://github.com/BeltranGSH/TallerPython.git
cd TallerPython
\`\`\`

2. **Crear entorno virtual (recomendado):**
\`\`\`bash
python -m venv venv
# En Windows:
venv\Scripts\activate
# En Mac/Linux:
source venv/bin/activate
\`\`\`

3. **Instalar dependencias:**
\`\`\`bash
pip install -r requirements.txt
\`\`\`

## Cómo Ejecutar el Código

### Ejecutar scripts de Python:

\`\`\`bash
# Script principal con ejercicios básicos
python src/main.py

# Ejemplos aplicados a ciencia/ingeniería
python src/ejemplos_ciencia.py
\`\`\`

### Abrir el cuaderno Jupyter:

\`\`\`bash
jupyter notebook notebooks/TallerPython_BeltranJesus.ipynb
\`\`\`

O abrir directamente en Google Colab usando el enlace del repositorio.

## Contenido del Taller

### 1. Entrada y Salida de Datos
- Uso de la función `input()`
- Conversión de tipos: `int()`, `float()`, `str()`
- Formato de cadenas con f-strings
- Captura y visualización de datos

### 2. Estructuras Básicas
- Variables y tipos de datos
- Operaciones aritméticas (suma, resta, multiplicación, división, módulo, potencia)
- Estructuras de control (`if`, `elif`, `else`)
- Bucles (`for`, `while`)
- Listas y operaciones con listas

### 3. Módulos Científicos
- **NumPy**: Arrays, operaciones matemáticas avanzadas
- **Matplotlib**: Visualización de datos y gráficas
- **Pandas**: Manipulación de datos tabulares

### 4. Ejemplos Prácticos
- Cálculos de cinemática (MRU, MRUA)
- Conversión de unidades
- Análisis estadístico de datos
- Visualización de funciones matemáticas
